# ProjetJavaScript
## Comment lancer le jeu ?
Il suffit de rentrer la commande "node jeu.js" dans le terminal en étant dans le dossier contenant ce fichier (en l'occurence le même où vous lisez ce fichier) afin de lancer l'éxecution du jeu. Toutes les instructions sont indiquées sur le terminal au lancement du jeu 

## Comment sont enregistrées les manches joueés ?
Au lancement du jeu, un fichier "resultats.txt" sera crée. Il contiendra les résultats de chaque joueur à chaque fin de tour et le gagnant de la partie lorsque cette dernière sera terminée. (ps : je suis resté sur un "format" synchrone pour l'écriture dans le fichier)

Bon jeu !
